# Gas-Leakage-Detection-Automatic-Shut-off-System
This project focuses on improving safety by detecting LPG gas leakage and automatically stopping the gas supply. An MQ gas sensor continuously monitors gas concentration in the surrounding area. When leakage exceeds a preset threshold, the system activates a buzzer alert and controls a relay module to close a solenoid valve, thereby shutting off the gas flow

The system helps prevent fire hazards, explosions, and gas wastage while demonstrating practical implementation of embedded systems and real-time automation.

🎯 Project Objectives:-

1)To detect LPG gas leakage using a gas sensor

2)To alert users through an audible buzzer during leakage.

3)To automatically shut off the gas supply using a solenoid valve.

4)To enhance domestic and industrial safety.

5)To gain hands-on experience in embedded systems and sensor interfacing

🧰 Hardware Requirements:-

•MQ Gas Sensor (MQ-2 / MQ-6)

•Arduino Uno / compatible microcontroller

•Relay Module

•Solenoid valve

•Buzzer

•Jumper Wires

•Breadboard

•Power Supply

💻 Software Requirements:-

•Arduino IDE

•Embedded C

•USB Driver for Arduino

⚙️ Working Principle:-

•The MQ gas sensor continuously senses LPG gas concentration in air.

•Sensor output is sent to the Arduino microcontroller.

•If the detected value crosses the predefined safety threshold:

•The buzzer is activated to alert users.

•The relay module switches ON.

•The solenoid valve closes automatically, stopping gas flow.

•When gas levels return to normal, the system resets.

•This process enables automatic safety control without human intervention.
